# kahvi's blog
Rebuild of my [Gatsby blog](https://github.com/iamkahvi/gatsby-blog) with Astro.

## Install and Usage
1.  **Install packages.**

    ```sh
    yarn
    ```
2.  **Start developing.**

    ```sh
    yarn run dev
    ```

## Built With
* [Astro](https://docs.astro.build/en/getting-started/)
* [Tachyons](https://tachyons.io/)
* [Contentful](https://www.contentful.com/)
